# dotfiles - and other config files for macOS

### 1. Install Homebrew

```
sh -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

### 2. Install all the things

- Brew packages
- Oh-My-Zsh
- Latest LTS Node version
- VS Code extensions
- VS Code settings
- dotfiles (zsh, misc, git)

```
sh setup.sh
```
